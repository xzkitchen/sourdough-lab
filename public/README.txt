# Public Assets

这个文件夹用于存放静态资源。

## 需要添加的文件

1. favicon.ico - 网站图标（16x16 或 32x32）
2. icon-192.png - PWA 图标（192x192）
3. icon-512.png - PWA 图标（512x512）
4. og-image.png - 社交媒体分享图（1200x630）

## 图标设计建议

- 使用面包、麦穗相关图案
- 背景色：#f97316（橙色）或 #0c0a09（深色）
- 简洁明了，易于识别

## 如何生成图标

1. 在线工具：https://realfavicongenerator.net
2. 上传一张 1024x1024 的PNG图片
3. 下载生成的图标包
4. 放到这个文件夹
